# Siri 唤起 ## 是什么 Siri 唤起是 Get笔记的语音快捷操作功能。设置好之后，你只需要对着手机喊一声 Siri，就能直接开始录音、写笔记或搜索，完全不用打开 App、不用动手。 特别适合开车、跑步、做饭等双手不方便的时候。 ## 怎么设置 ### 第 1 步：打开 Get笔记 App 打开 Get笔记，点击右上角的头像，进入个人中心。 ### 第 2 步：找到快捷指令入口 在个人中心页面，找到「Siri、操作按钮等快捷指令」，点击进入。 ### 第 3 步：添加你需要的快捷指令 页面会列出所有可用的快捷指令，点击每个指令右侧的「+」按钮，就能添加到 Siri。 ![Get笔记添加 Siri 快捷指令页面，显示「开始录」「去开会」「问助手」「搜笔记」「写笔记」五个可添加的快捷指令，每个指令右侧都有加号按钮](/getbiji-docs-v3/images/PRg5bAEe3oUyMUxKkpCc9QkqnXd.png) ### 第 4 步：对 Siri 说出指令 添加成功后，随时对 Siri 说出对应的触发词就能使用了。 ## 支持的指令 | 对 Siri 说 | 会做什么 | |-----------|---------| | 「开始录」 | 快速启动语音笔记，开始录音 | | 「去开会」 | 启动会议录音笔记，自动记录会议内容 | | 「问助手」 | 打开 AI 助手，直接提问 | | 「写笔记」 | 进入文本笔记编辑界面 | | 「搜笔记」 | 打开搜索页面，查找笔记 | ## 什么时候用 - **开车途中**突然有灵感 → 喊「嘿 Siri，开始录」 - **进会议室前** → 喊「嘿 Siri，去开会」 - **运动时**想记点什么 → 喊「嘿 Siri，开始录」 - **做饭时**想查笔记 → 喊「嘿 Siri，搜笔记」 ## 小技巧 - 你可以根据自己的习惯自定义触发词，比如把「开始录」改成「记一下」，用起来更顺口 - 在安静的环境下使用语音指令，Siri 识别更准确
